# Deploy Now - Documentation

Use [Gridsome](https://gridsome.org) with NodeJS v12.22

## Setup

Install dependencies and edit new `.env` file
```bash
npm ci
cp .env.example .env
```

## Develop

Run local server
```bash
npm run develop
```

Generate content pages with cli
```bash
npm run cli
```

## Build

Create static build in `dist` folder
```bash
npm run build
```
